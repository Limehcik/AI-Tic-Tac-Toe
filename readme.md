### [🤖 Спецификация ONNX](HowUse.md)

# Установка зависимостей
Скачать зависимости `pip install -r requirements.txt`
> (При наличии видеокарты NVIDIA автоматически скачается версия Torch с поддержкой CUDA)

# Запуск
Запуск происходит через файл `main.py`

+ Для сброса поля игры нажмите `Пробел`

# Анализ
Файл `analyz.py` Проводит анализ модели в формате *.liai* и выводит полную информацию и результаты теста в разных ситуациях у этой модели
> Полезно при диагностике модели или финальной проверке перед экспортом

# Экспорт в ONNX
Для экспорта запустите файл `convert_to_onnx.py`, на выходе вы получите модель в расширении *.onnx*, подходящем для любой иной программы/движка/сайта
